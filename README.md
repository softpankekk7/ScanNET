## ScanNet

**ScanNet** is a reconnaissance and host discovery tool intended for **authorized penetration testing, security research, and network auditing**.

This software performs automated host discovery and port probing across IP ranges. These actions are commonly classified as **reconnaissance activity** and may be interpreted as malicious by intrusion detection systems, firewalls, ISPs, and service providers.

**Running this tool against networks, systems, or devices without explicit authorization may be illegal in many jurisdictions.**

By using this software, you agree that:

- You will only use ScanNet against systems you **own** or have **explicit written permission** to test.
- You understand that port scanning and host enumeration may trigger alerts, bans, or legal investigation.
- You accept full responsibility for how you use this tool.
- The author(s) are **not responsible** for misuse, damages, service disruption, or legal consequences.

**Do not use this tool if you do not have permission.**

---

## Intended Use

ScanNet is designed for:

- Penetration testing labs
- Red team exercises
- Security research
- Network auditing
- Learning how reconnaissance techniques work
- Internal infrastructure mapping

It is **not** intended for unauthorized internet-wide scanning or opportunistic probing.
